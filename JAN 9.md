After executing the following code snippet, which of the options does NOT produce: array([1, 2, 3, 4, 5, 6])

import numpy as np

a = np.array([[1,2],[3,4],[5,6]])

A: a.flatten()

B: a.ravel()
 
C: a.reshape(-1)

D: a.unravel()

Answer:- Option D: a.unravel()

Explanation:- 

There is no such method as unravel present in python
